# Brent Oil Simulator

A full-stack simulation platform that models Brent crude oil price movements over 6 months based on Middle East geopolitical scenarios.

## 🚀 Live Deployment (Recommended)

Deploy instantly using Vercel:

1. Go to https://vercel.com
2. Import this GitHub repo
3. Click Deploy

Vercel will automatically detect Next.js.

## 🛠 Local Development

```bash
npm install
npm run dev
```

## ⚙️ Features
- Scenario-based oil price simulation
- API-driven architecture
- Frontend + backend in Next.js

## 🧠 Simulation Factors
- War escalation level
- Supply disruption
- OPEC production decisions
- Market randomness
